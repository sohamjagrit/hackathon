# Hotel front desk — demo script (~25 seconds)

You answer the **hotel** phone when Miles calls mid-recovery.

## What to do
1. Let it ring once, pick up.
2. Agent says they’re calling about guest **[name]** / PNR **[code]**.
3. They ask to change dates (shorter stay / new check-in).
4. Ask **one** question only, e.g. “Same room type?”
5. Say: **“Yes — I’ve updated the reservation for those dates. You’re all set.”**
6. Hang up.

## What not to do
- Don’t talk payment, loyalty, or upgrades.
- Don’t put them on a long hold.
- Don’t refuse the change (unless rehearsing a failure take).

A little natural friction is fine. Leisurely chat kills the traveler’s hold pacing.
